Aim
To study the Pandas library in Python and perform operations such as creating Series and DataFrames, 
accessing rows and columns, modifying data, deleting columns, and performing basic statistical analysis.

Theory
Pandas is an open-source Python library used for data manipulation and data analysis. 
It provides powerful data structures like Series and DataFrame which help in handling structured data easily.

Series
A Series is a one-dimensional labeled array capable of holding data of any type.

DataFrame
A DataFrame is a two-dimensional table-like structure consisting of rows and columns.

DataFrame Properties
df.ndim → Number of dimensions
df.size → Total number of elements
df.shape → Number of rows and columns
df.columns → Displays column names
df.dtypes → Displays data type of each column

Statistical Analysis
Pandas provides built-in functions such as:
mean() → average value
min() → minimum value
max() → maximum value
Viewing Data
head() → displays first 5 rows
tail() → displays last rows

Conclusion
Thus, we successfully studied the Pandas library and learned how to create Series and DataFrames, access and modify data, delete columns, perform statistical analysis, and filter data. Pandas provides efficient tools for handling and analyzing structured data in Python.
